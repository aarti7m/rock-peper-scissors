# rock-peper-scissors
This project is a classic Rock-Paper-Scissors game where the player selects Rock, Paper, or Scissors and the computer randomly generates its move. The result updates instantly with images and score tracking. Designed to be simple, responsive, and fun.
Features:
1.  Player vs Computer
2.  Scoreboard Display
3.  Interactive Image Buttons
4.  Clean and beginner-friendly UI
